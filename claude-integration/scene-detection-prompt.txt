**Aufgabe:**
Analysiere die SRT-Datei und identifiziere interessante Szenen für Social Media Clips (Podcast-Teaser, Reels, Shorts).

**Kriterien:**
* **Länge:** Mindestens 30 Sekunden, maximal 90 Sekunden
* **Inhalt:** Suche nach **vollständigen Gedankengängen**, die einen Gedanken entwickeln:
  - Ein Argument mit Begründung oder Beispiel
  - Eine These mit Entfaltung oder Kontext
  - Eine Geschichte mit Anfang und Schluss
  - Ein Konzept mit Erklärung
  - Dialog-Szenen mit mehreren Sprechern (besonders lebendig!)
  - Frage-Antwort-Sequenzen mit Ausführung
* **Qualität:**
  - Vollständige Gedankengänge, nicht nur einzelne Aussagen
  - Der Gedanke muss sich entwickeln und abschließen
  - Keine abgebrochenen Sätze oder Fragmente

**Wichtig:** Finde 8-12 Szenen. Bevorzuge Szenen mit Dialog oder ausgeführten Gedankengängen.

**Ausgabeformat:**

Gib ZUERST eine kurze Analyse im Chat aus (was du gefunden hast), und dann **am Ende deiner Antwort** ein JSON-Code-Block mit diesem exakten Format:

```json
{
  "mode": "manual",
  "selection_type": "time",
  "clips": [
    {
      "name": "Kurzer beschreibender Titel der Szene",
      "start_time": 23.5,
      "end_time": 78.2
    }
  ]
}
```

**Wichtig:**
- Zeiten in **Sekunden** (float), nicht HH:MM:SS
- `start_time` und `end_time` müssen exakte Werte aus der SRT-Datei sein
- Jeder Clip sollte 30-90 Sekunden lang sein
- Der JSON-Block muss am Ende deiner Antwort stehen
